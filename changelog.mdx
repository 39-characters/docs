<Update label="2026-03-15" description="March 2026" tags={["Feature", "Update"]}>
  ## AI-powered documentation workflows

  You can now use your favorite AI coding tools to write and maintain Night City documentation. Setup guides are available for [Cursor](/ai-tools/cursor), [Claude Code](/ai-tools/claude-code), and [Windsurf](/ai-tools/windsurf). Each guide walks you through installing the Mintlify skill and configuring your editor for documentation workflows.

  ## Contextual code actions

  Documentation pages now include contextual actions that let you open code examples directly in Cursor, VS Code, ChatGPT, Claude, Perplexity, or via MCP. Look for the action menu on any code block to use it in your preferred tool.

  ## Plants API

  The Plants API is live with endpoints for [listing plants](/api-reference/endpoint/get), [creating plants](/api-reference/endpoint/create), [deleting plants](/api-reference/endpoint/delete), and receiving [webhook notifications](/api-reference/endpoint/webhook) when new plants are added. All endpoints use Bearer token authentication.

  ## Dropdown navigation

  The site navigation now uses a dropdown layout, making it easier to switch between Guides and API Reference sections. You can learn more about configuring navigation in the [navigation guide](/essentials/navigation).
</Update>
